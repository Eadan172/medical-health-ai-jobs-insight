# medical-health-ai-jobs-insight
无代码可视化网页，用于分析51job、boss、猎聘上岗位信息，岗位方向面向医药健康+AI。
###每日0点更新###

🎉 可直接访问的网站
访问地址: https://fpbec7iecuz4g.ok.kimi.link

📊 功能亮点
1. Hero主视觉区
动态渐变背景 + 粒子动画效果
实时数据概览（500+岗位、3大平台、10+城市）
2. 平台生态概览
三大招聘平台（51job、Boss直聘、猎聘网）数据对比卡片
3D悬停效果，展示各平台的岗位数量和平均薪资
3. 薪资分布洞察
平台平均薪资对比柱状图
薪资区间分布环形图（15-25K、25-35K、35-50K等）
4. 地域机会分布
面积图展示各城市岗位数量与平均薪资
覆盖北京、上海、广州、深圳、杭州等10大城市
5. 核心技能图谱
热门技能需求排行（Python、TensorFlow、医学影像、深度学习等）
进度条动画展示技能需求占比
6. 经验要求分析
横向柱状图展示不同经验段的岗位分布与薪资
7. 精选岗位推荐
支持按平台和城市筛选
展示岗位详情、公司、薪资、技能要求等
可展开/收起查看更多岗位

🎨 设计特色
深色科技风：深蓝底色（#0A1628）搭配科技蓝（#1B45F4）
流畅动画：使用Framer Motion实现滚动触发动画
交互丰富：悬停效果、筛选功能、数据可视化
响应式设计：适配桌面和移动设备
🛠 技术栈
React + TypeScript + Vite
Tailwind CSS
Recharts（数据可视化）
Framer Motion（动画效果）
